## [ranger](https://wiki.archlinux.org/title/ranger)
ranger is a text-based file manager written in Python with vi-style key bindings.

```
$ pacman -S ranger
$ ranger --copy-config=all
```

Show hidden file
```
:set show_hidden true
```
